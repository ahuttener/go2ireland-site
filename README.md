# go2ireland.site

Site estático da **Keymate** para a Irlanda — quartos e roommates com compatibilidade.

- 17 páginas HTML + assets (`img/`, `logo.png`, `og-image.png`)
- Sem backend: interações (busca, filtros, upload de foto de perfil, mapa) rodam no navegador
- Mapa real via Leaflet + OpenStreetMap/CartoDB

## Deploy

Hospedado na **Hostinger**, servido a partir da branch `main` deste repositório.
Qualquer `push` na `main` atualiza o site (basta a Hostinger reimplantar).
